## Rule
1. 요일 : 월, 수, 목
2. 문제 : BOJ 기준
    - 목 PT 후 차주 문제 3개 선정
3. 언어 : Java

## PR 규칙 및 Commit Message 규칙

#### Pull Request
- [BJ_폴더명_문제수] 이름

#### Commit Message
- [BJ_문제번호] 문제명
